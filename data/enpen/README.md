# Nextclade datasets - ENPEN collection

This directory contains source files of Nextclade datasets comprising Nextclade ENPEN dataset collection.

These datasets are created and maintained by ENPEN members. Nextclade team cannot provide support for these datasets or to guarantee their correctness. Please reach out to dataset authors for all details.
